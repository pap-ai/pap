# /new-workspace

Triggers the curiosity agent to start an intake
interview for a new ongoing deliverable.

If invoked with no description: opens an intake
conversation in #new-workspace.

If invoked with brief description: starts the
interview with that as initial context.

This is a shortcut for power users — most users
just type their idea in #new-workspace and the
dispatcher routes accordingly.
