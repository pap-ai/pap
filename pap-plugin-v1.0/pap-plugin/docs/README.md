# Personal Automation Platform (PAP)

> Your AI Chief of Staff coordinates a team of specialists that works around the clock — even while you sleep.

PAP is an always-on AI agent system built on Claude's Cowork platform. It automates real work, runs while you sleep, and gets smarter about what you want over time.

**Full documentation:** https://pap-ai.github.io/pap

## Quick install

In Claude Desktop → Cowork tab → Customize → Add Plugin:
```
github.com/pap-ai/pap
```

## What you need

- Claude Pro subscription ($20/month)
- A dedicated always-on computer (old laptop, Mac Mini, etc.)
- Discord account
- GitHub account (for backups)
- 30-45 minutes for initial setup

## Two modes

**Mode 1 — Dedicated clean machine (strongly recommended)**
Full PAP experience. Runs 24/7. Security isolated from your personal data.

**Mode 2 — Your daily machine**
Works, but Computer Use has access to your personal files and accounts. Scheduled tasks stop when machine sleeps.
